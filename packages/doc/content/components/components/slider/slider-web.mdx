### Reference

Gympass `<Slider />` component extends from <a href="https://github.com/react-component/slider/" target="_blank" rel="noopener noreferrer">rc-slider</a>.

### Usage

#### Simple slider

```javascript state
render(() => {
  const [values, setValues] = useState([0]);

  return (
    <Slider
      values={values}
      onChange={value => setValues([value])}
      max={10}
      min={0}
      minLabel={values[0]}
      tooltip={[
        {
          description: 'Simple tooltip',
          step: 2,
          visible: true,
        },
      ]}
    />
  );
});
```

#### Showing a tooltip

Is possible to render a tooltip over the marks of slider, to do that you need to pass the `tooltip` prop. It's takes an array of objects that accepts the current shape:

- ribbon: `string`
- title: `string`
- description: `string`
- visible: `bool` (condition that controls tooltip visibility)
- step: `number` (a specific step to show the tooltip)
- alwaysShow: `bool` (if the tooltip would be visible, it will keep showing even after the release event)

With this array of objects you can render different tooltips, with different messages for each respective step you like.
