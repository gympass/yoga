GuidelinesComponentsSystem

    Reference

    Gympass <Button /> component implements all our design guidelines such as default spacings, borders and color pallete tokens. You can use all differente compounds like <Button.Outline /> and <Button /> combining with their props such as small or inverted so the button become the desired variation.

    Usage

    Default

    import { Button } from '@gympass/yoga';
    <>
      <Button.Text>Find an activity</Button.Text>
      <Button.Text secondary>Find an activity</Button.Text>
    </>

    With icon

    import { Booking } from '@gympass/yoga-icons';
    import { Button } from '@gympass/yoga';
    <Button.Text icon={Booking}>Find an activity</Button.Text>
