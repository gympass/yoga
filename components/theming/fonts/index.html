    Fonts

    Fonts can define our brand and the way we express ourselves through titles and texts.

    Usage

    This is a simple usage:

    import styled from 'styled-components';
    import { theme } from '@gympass/yoga';
    const MyCoolComponent = styled.div`
      font: ${theme.fonts.rubik};
    `;
    Theme Family Weight
