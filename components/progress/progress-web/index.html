GuidelinesComponents

    Usage

    import { Progress } from '@gympass/yoga';
    <div style={{ width: 500 }}>
      <Progress max={100} value={65} label={{ value: 65, placement: 'right' }} />
    </div>

    Examples


    import { Progress } from '@gympass/yoga';
    <div style={{ display: 'flex', flexDirection: 'column', width: 500 }}>
      <Progress max={100} value={20} label={{ value: 20, placement: 'left' }} />
      <Progress max={100} value={40} label={{ value: 40, placement: 'left' }} />
      <Progress max={100} value={67} label={{ value: 67, placement: 'left' }} />
      <br />
      <Progress
        max={100}
        value={70}
        label={{ value: 'Some description', placement: 'left' }}
      />
    </div>

    Props

    PropDescriptionTypeDefaultRequired
    labelUse labels to increase users understanding. If the value is numeric, make sure it has a maximum of 3 charactersshape{ value: undefined, placement: left, }false
    maxThis attribute describes how much work the task indicated by the progress element requires. The max attribute, if present, must have a value greater than zero and be a valid floating point number. The default value is 1.number1false
    valueThis attribute specifies how much of the task that has been completed. It must be a valid floating point number between 0 and max, or between 0 and 1 if max is omitted. If there is no value attribute, the progress bar is indeterminate; this indicates that an activity is ongoing with no indication of how long it is expected to take.numberundefinedfalse
