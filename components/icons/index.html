    Icons

    Yoga provide a package called @gympass/yoga-icons that contains icons to be used at your application.

    Installation

    yarn add @gympass/yoga-icons
    • Web
    • Native

    Usage

    Icon components recieve all the available svg tag attributes.

    import { Star } from '@gympass/yoga-icons';
    <Star stroke="#ffc24c" width={26} height={26} />
