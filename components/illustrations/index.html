    Illustrations

    Yoga provide a package called @gympass/yoga-illustrations that contains illustrations to be used at your application.

    Installation

    yarn add @gympass/yoga-illustrations
    • Web
    • Native

    Usage

    Illustration components receive all the available svg tag attributes.

    import { Img2d01 } from '@gympass/yoga-illustrations';
    <Img2d01  width={375} height={375} />
